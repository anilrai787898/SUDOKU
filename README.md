🎮 Sudoku Game
A clean, fast, and fun Gen Z-styled Sudoku web app with high score tracking — built with vanilla HTML, CSS, and JavaScript.

✨ Features

🧩 Auto-generated Sudoku puzzles — new game every time
🏆 High score persistence — scores saved via localStorage
⏱️ Timer — tracks how fast you solve the puzzle
🎨 Gen Z aesthetic — modern UI with clean styling
✅ Input validation — highlights errors in real time
📱 Responsive design — works on mobile and desktop
🔄 New Game / Reset — start fresh anytime
